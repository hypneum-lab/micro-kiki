Quel signal circule sur le nerf entre WMLs ?

Ta formule : data → nerf → WML. Un WML = un ensemble de neurones. Mais le « data » porté par le nerf peut prendre trois formes très différentes — chacune mène à une architecture radicalement distincte.

A

Spikes — Potentiels d'action discrets

Analogie biologique : axone qui tire ou ne tire pas. Codage par le timing et la fréquence, pas l'amplitude.

WMLA ══╪═╪══╪═══╪═╪══► WMLB
          t₀   t₁     t₂   t₃
Binaire · timing-encodé · sparse

Implication : SNN/LIF, LAS-converti, bande passante ultra-basse, énergie minimale. Latence dominée par le taux de tir.

B

Vecteurs latents — Hidden states continus

Analogie biologique : potentiel gradué (rétine, cellules amacrines) — amplitude continue, pas de seuil tout-ou-rien.

WMLA ══ [ 0.42, −0.18, 0.07, …, 0.91 ] ══► WMLB
Dim fixe (ex. 128) · dense · différentiable

Implication : comme le KikiFlowBridge (kiki-flow-research) ou Aeon Latent Predictor. Bande passante élevée, gradient-friendly, facile à entraîner.

C

Tokens abstraits — Alphabet appris non-linguistique

Analogie biologique : neurotransmetteurs — un petit vocabulaire discret (GABA, glutamate, dopamine, sérotonine…) qui module le récepteur post-synaptique.

WMLA ══ [ γ₇ · α₂ · ε₃ · … · β₁₁ ] ══► WMLB
Vocabulaire de ~16-256 « neurolettres » · discret · compositionnel

Implication : VQ-VAE / discrete bottleneck entre WMLs. Compositionnel, inspectable, mais non-différentiable (besoin STE/Gumbel).

Tu peux aussi répondre autre chose dans le terminal — par exemple un hybride (spikes + amplitude variable, ou latents + quantisation légère).
