Qu'est-ce qu'un WML, concrètement ?

Tu as dit « WML = un ensemble de neurones ». Trois granularités possibles, chacune menant à un projet d'ampleur très différente — du plug-in dans micro-kiki à un nouveau moteur neuro-symbolique standalone.

A

Stack LoRA — Adapter domain-expert

Taille : ~10-100 M params (rank 8-32 sur Qwen 3.5-35B). Où ça vit : directement dans micro-kiki/src/stacks/, un WML = une spécialisation.

WMLkicad ═══γ π═══► WMLspice
WMLkicad ◄══θ ε═══ WMLspice
(chaque WML = 1 LoRA adapter sur Qwen3.5-35B-A3B partagé)
35 WMLs potentiels · infrastructure existante · impact production

Atout : branche immédiatement sur ton runtime. Atout 2 : les neurolettres π/ε sont apprises par-dessus des experts déjà compétents. Limite : les WMLs partagent le backbone 35B — la diversité des « neurones » est simulée, pas physique.

B

Micro-MLP dédié — Module indépendant

Taille : ~100 k à 10 M params chacun. Où ça vit : nouveau moteur (nouveau repo ou sous-projet dreamOfkiki/kiki-flow-research), pas sur un LLM.

┌─WMLA─┐    ┌─WMLB─┐
│MLP 4×256│━━► nerf ━━►│MLP 4×256│
│embed │◄━━ nerf ━━━│embed │
└─────────┘    └─────────┘
PoC propre · pas de LLM · entraînable sur GrosMac

Atout : clean slate — pas de dépendance Qwen/LoRA. Atout 2 : permet d'étudier le protocole de nerf en isolation (paper-friendly). Limite : pas de branchement direct avec la production LLM ; ambition d'une plateforme séparée.

C

Groupe de neurones — Population LIF/SNN

Taille : ~50-500 neurones biologiquement plausibles (LIF, Izhikevich), chacun avec son état membraire. Où ça vit : nouveau moteur neuromorphique, inspiré de Nengo / Brian2 / ta branche SpikingKiki v0.3.

┌──WMLA──┐    ┌──WMLB──┐
│ 100 LIF │──axone γ──►│ 100 LIF │
│ neurons │◄──axone θ──│ neurons │
│ Vm dyn │            │ Vm dyn │
└───────────┘       └───────────┘
Pure neuro · énergie mesurée en spikes/s · paper neurosciences

Atout : le plus fidèle à « ensemble de neurones ». Atout 2 : mesurable énergétiquement, déployable sur hardware neuromorphique (Loihi, Akida). Limite : non-différentiable (surrogate grad ou STDP), entraînement plus lent, intégration LLM lointaine.

Dernière question après : taille de l'alphabet de neurolettres (16 · 64 · 256 · appris). Puis je propose 2-3 approches architecturales complètes avant de rédiger le design.
